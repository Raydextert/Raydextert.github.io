a标签引入脚本以及layer插件的使用

Posted on 2019-11-28

关于html中a标签传入脚本的几种方式

1、直接在href属性中添加方法形如：< a href=”javascript:js_method()”>javascript只是一个协议，后面紧跟着的是一个调用方法，注意调用的是方法，而不是从外部引入的脚本文件。
2、在a标签中的onclick事件中绑定方法，形如：< a href=”javascript:void(0)” onclick=”method()”>void是一个操作符，void(0)则表示的是地址不发生跳转。
3、形如< a href=”#” onclick=”js_method();return false;”>执行了js之后页面不发生跳转
注：以上的js都指的是的方法而不是js文件

layer的使用方法

function modifInfosFrm() { 
layui.use('layer',function()
{
   var layer=layui.layer;
   layer.open({
     type:2,
     area:['600px','330px'],
     shadeClose:true,
     title:['个人信息','font-size:18px'],
     content: 'page/userInfos.html'
   });

});

}

当要将html文件作为内容传入layer窗体中时 type类型要为2content中的可以填绝对路径也可以填相对路径
另外，layui.use是layer作为模块时的调用方法，当他作为独立组件的时候直接调用layer.open函数就行了
